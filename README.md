# BMI Kids Review

## Authors:

**Steven B. Heymsfield**, David A Hughes, Christian Rodriguez, Sri Lakshmi S. Devarakonda, Shengping Yang, Diana M. Thomas, Angelo Pietrobelli

## Description

This is code used to run analyses and generate figures for a review paper on the use of BMI in kids.

## Data

All data is derived from National Health and Nutrition Examination Survey or [**NHANES**](https://wwwn.cdc.gov/nchs/nhanes/continuousnhanes/default.aspx?Cycle=2021-2023).
